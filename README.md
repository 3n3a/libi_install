# Mmhh, screenshots...

## Installation

```cmd
curl <url>/raw/installer.bat -L -o installer.bat
installer.bat
```

## Usage

### Prepare iPhone

For this to work iTunes needs to be installed (Download [Win64](https://www.apple.com/itunes/download/win64))!

```cmd
prepare_iphone.bat
```

### Take Screenshot

```cmd
screenshot.bat
```

The screenshot can now be found on your Desktop.
